brew install node

npm install -g hexo

DNSpod配置

1.这里的下图被马赛克的其实就是github的IP：192.30.252.153，意思就是papp.life域名解析到github的IP，因为我们的博客放在github的仓库。

去GoDaddy里面加入DNSpod的记录

2.这边大概的意思就是去我们域名提供商那把我们域名配置用DNSpod解析，再加上上一步github的IP，这样就把域名提供商、DNSpod、github3者的关系建立起来了。(还需要第三步，在仓库加入CNAME文件)

配置github

3.然后还要在我们的blog的git仓库里面放一个CNAME文件。新建一个CNAME文件里面写上域名就行了。（这里最好放在HEXO文件目录下的source文件夹，这样不会每次部署的时候被覆盖，只要配置一次就好了。）
